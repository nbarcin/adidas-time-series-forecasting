# 📈 Adidas Time Series Forecasting

This project analyzes Adidas quarterly sales revenue using time series analysis and forecasting techniques. The notebook explores historical revenue trends, seasonal patterns, and future sales predictions using SARIMAX modeling.

The project demonstrates the complete workflow of a time series forecasting project, including preprocessing, visualization, decomposition, model building, and future forecasting.

---

# 📌 Project Overview

Time series forecasting plays a major role in business analytics, financial analysis, and sales prediction. In this project, Adidas quarterly revenue data was analyzed to:

- Understand sales trends over time
- Detect seasonality and patterns
- Build a forecasting model
- Predict future quarterly revenue

The project uses classical statistical forecasting techniques to model time-dependent behavior.

---

# 📊 Dataset Information

The dataset contains Adidas quarterly sales revenue data collected over multiple time periods.

Main features include:

- Quarter
- Revenue
- Time index

The dataset was transformed into a time-indexed format suitable for forecasting analysis.

# 🔍 Methodology

## 1. Data Preprocessing

The dataset was prepared by:

- Converting date columns into datetime format
- Setting time-based index
- Sorting chronological data
- Handling missing values if necessary

---

## 2. Exploratory Time Series Analysis

The project includes:

- Revenue trend visualization
- Seasonal pattern analysis
- Time-based plotting
- Rolling averages

These visualizations help identify long-term trends and seasonal effects.

---

## 3. Seasonal Decomposition

Seasonal decomposition was applied to separate:

- Trend component
- Seasonal component
- Residual component

This helps better understand the underlying structure of the time series.

---

## 4. SARIMAX Forecasting Model

A SARIMAX model was used to forecast future quarterly revenue values.

The model captures:

- Autoregressive patterns
- Seasonality
- Trend behavior

Future revenue predictions were generated based on historical sales patterns.
